# Asset Sources and Licensing

## Bundled assets

Version 0.10.0-alpha.0 bundles only original, code-authored media: SVG projectile and impact graphics in `assets/vfx`, 20 BattleMech portraits, 44 item icons, six vehicle portraits, 26 synthesized PCM WAV activation cues, and procedural weapon tones generated at runtime. The portraits, icons, and WAV files are deterministically generated by `tools/generate-mech-media.mjs`. No third-party image or audio files are bundled.

## Optional upstream integrations

- Dice So Nice (`riccisi/foundryvtt-dice-so-nice`) is AGPL-3.0. No Dice So Nice code or assets are copied. BMFS uses Foundry's standard Roll/chat pipeline, which Dice So Nice officially observes when separately installed.
- Tokenizer (`MrPrimate/tokenizer`) is MIT. BMFS calls its documented `tokenizeActor` API when the separately installed module is active; no Tokenizer code or frames are bundled.
- JB2A Free (`Jules-Bens-Aa/JB2A_DnD5e`) is CC BY-NC-SA 4.0. BMFS calls its registered Sequencer database paths when separately installed; no JB2A animation files are bundled.
- Token Action HUD PF2E (`Larkinabout/fvtt-token-action-hud-pf2e`) is CC BY 4.0 and is Pathfinder-specific. BMFS implements an original, dependency-free BattleTech HUD based on the general interaction concept; no PF2E code, translations, or styles are copied.
